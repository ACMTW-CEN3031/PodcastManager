Sequential Artists Workshop Web Interface


About

The Sequential Artists Web Interface is designed to help create exercises for teachers and students for comic ideas, activities and exercises.

If you're not logged in click Sign in or Sign up to get started.

After you've logged in you can:
Create a Deck, which allows you to upload a premade image set.
View decks to preview or edit prexisting decks.
Make a Spread using prexisting decks,
or collaborate using the Message Board.
